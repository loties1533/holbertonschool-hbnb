# HBnB (Holberton BnB)

## 🌍 Description

HBnB est un projet full-stack inspiré d’Airbnb, développé dans le cadre du programme de la Holberton School.
L’objectif est de créer une plateforme complète permettant la gestion de logements, d’utilisateurs et de réservations, tout en explorant les notions clés du développement web, du stockage de données et du déploiement d’applications.

## ⚙️ Structure du projet

Le projet est conçu en plusieurs étapes (ou “phases”) :

  1. Console – un interpréteur de commandes en Python pour gérer les objets et leurs relations.
  2. Web Static – intégration des premières pages HTML/CSS.
  3. Web Framework – implémentation d’un serveur web Flask affichant les données dynamiquement.
  4. RESTful API – création d’une API pour communiquer entre le front-end et la base de données.
  5. Database & Storage – utilisation de MySQL pour le stockage persistant et gestion multi-environnements (FileStorage / DBStorage).
  6. Deployment – configuration de l’application pour être servie via un serveur Nginx et Gunicorn avec Docker.

## 🧠 Technologies utilisées
Python 3
Flask (framework web)
MySQL
HTML / CSS
API REST
Unit tests / PEP8 compliance

## 🚀 Objectifs pédagogiques

  - Comprendre l’architecture d’une application web complète
  - Concevoir et interconnecter les différentes couches : back-end, front-end et base de données
  - Développer de bonnes pratiques de versionnage, de documentation et de tests
